# Бакалаврська робота | Автомобільна аналітика 🎓

Цей репозиторій містить офіційний лендінг для презентації бакалаврської роботи на тему: **«Розробка кросплатформного застосунку для автомобільної аналітики»**.

🌐 **[Переглянути готовий веб-сайт](https://github.com/Dima35954/bachelor-landing)**

## Про проєкт
Веб-сайт розроблено з дотриманням сучасних вимог до веб-розробки:
- **Семантика та Доступність:** Використання HTML5 тегів та ARIA-атрибутів.
- **SEO-оптимізація:** Налаштовані метатеги, Open Graph для соцмереж та мікророзмітка Schema.org.
- **Адаптивність:** Коректне відображення на мобільних та десктопних пристроях завдяки чистому CSS (без сторонніх бібліотек).

## Стратегія розробки
Проєкт створено з використанням системи контролю версій Git за стратегією **Trunk-Based Development (TBD)**. Історія комітів та тегів версій детально відображає поетапний процес імплементації технічних вимог.
